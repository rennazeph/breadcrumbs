# Backpack v1.31.2

Template - Medium was reworked into a three-column wiki stress test.

## Changes

- Replaced the old Description/Families medium template with a registry-driven wiki page model.
- Added Column A cascade navigation for Wiki Index, Template Overview, and bee article pages.
- Added Column B page rendering with title, description, and multiple Markdown subtitle fragments.
- Added Column C wiki-style image holder plus a smaller side-note Markdown import.
- Added multi-file Markdown bundle upload mapped by filename.
- Removed Description/Families copy buttons from the medium template UI.
- Added richer bundled bee demo content and source Markdown files under `content/wiki/`.
- Added additional bundled SVG placeholders for flower, food, and gallery wiki pages.

## Validation checklist

- Open Template - Medium.
- Click each cascade navigation entry.
- Upload several files from `content/wiki/` with the MD bundle input.
- Upload a side note for the current page.
- Upload an image for the current page.
- Export state and reload the page to verify uploaded wiki content persists.
