# Backpack 1.30 Release Notes

This release labels the current Backpack baseline as **1.30**.

## Major systems in this baseline

- Split HTML/CSS/JS build plus single-file fallback.
- Full-state export/import.
- Calendar-only export/import and maintenance tools.
- Quick Notes board-only export/import.
- Placeholder Settings and path derivation.
- Theme, reading-theme, density, and accessible-mode controls.
- Notes Markdown import display.
- Template - Basic and Template - Medium examples.
- CSS Museum instructional examples.

## Calendar highlights

- Start Day arrival flow.
- Plain-text arrival input with quick normalization.
- Mouse-friendly minute adjustment controls.
- Arrival logged audit state with hidden edit controls.
- Structured event fields.
- Gantt schedule using JS-measured pixel layout.
- Repeat this month event copies.
- Calendar diagnostics and repair tools.
- Grayscale-friendly state language for today/selected/hover/priority.

## Quick Notes highlights

- Editable note title and body.
- Movement from bottom chrome.
- Grid snapping.
- Controlled overlap tolerance.
- Text color and background color controls.
- Board export/import.

## Documentation cleanup

The README now reflects the current app rather than a chronological patch log. Detailed section docs live in `docs/`.
